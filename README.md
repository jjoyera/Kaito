# Kaito

Aplicación web con un coach de IA especializado en carreras de ultradistancia (trail, backyard, OCR). 
El sistema parte de una base de conocimiento de entrenadores reales para generar planes personalizados. 
Mediante un onboarding inicial evalúa el estado del corredor y su objetivo de carrera, y genera un plan de entrenamiento adaptativo. 
Conectado a Strava, recoge los datos reales de entrenamiento y ofrece un dashboard con métricas, calendario y un mapa de cumplimiento estilo GitHub. 
Si el corredor falla una sesión, el plan se recalcula automáticamente.
